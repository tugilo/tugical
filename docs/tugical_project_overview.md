# tugical（ツギカル）

次の時間が、もっと自由になる。

**統合 Laravel アプリケーション版 v2.3**

---

## プロジェクト概要

- サービス名：**tugical**（小文字で統一）
- スローガン：**次の時間が、もっと自由になる。**
- **統合 Laravel アプリケーション**（frontend + backend + liff → 単一アプリケーション）
- LINE 連携型 予約管理 SaaS（個人店〜多店舗対応）
- ブランド構成：
  - アイコン：t 入りカレンダー
  - 書体：Nunito
  - カラー：ミントグリーン × ダークグレー

### 🎉 統合完了の成果

**統合前:**

```
├─ frontend/ (別サーバー)
├─ liff/ (別サーバー)
├─ backend/ (Laravel)
└─ docs/ (ルート)
```

**統合後:**

```
└─ backend/ (統合Laravelアプリケーション)
    ├─ resources/js/ (統合フロントエンド)
    │   ├─ components/admin/ (管理者機能)
    │   ├─ components/liff/ (LIFF機能)
    │   ├─ pages/admin/ (管理者画面)
    │   ├─ pages/liff/ (LIFF画面)
    │   ├─ stores/ (状態管理)
    │   ├─ services/ (API)
    │   └─ utils/ (ユーティリティ)
    ├─ docs/ (統合ドキュメント)
    └─ package.json (統合依存関係)
```

### 統合のメリット

- **開発効率向上**: 単一リポジトリ・統合ビルド・統合テスト
- **運用効率向上**: 単一コンテナ・統合ログ・リソース最適化
- **保守性向上**: コード共有・型安全性・統合デプロイ

## 解決する課題と提供価値

### 現場課題

- 電話・紙台帳での予約管理
- ダブルブッキング／無断キャンセル
- 高機能予約ツールの導入ハードル

### tugical の提供価値

- LINE 完結 → 利用者側にやさしい
- **直感的操作** → ドラッグ&ドロップで誰でも簡単予約
- **モダン UI** → スマホ・PC 問わず美しく快適な操作感
- テンプレ選択で導入簡単
- 柔軟な予約条件設定（性別・設備等）

## 管理者機能（MVP）

- 店舗／施設／スタッフ管理
- メニュー登録（所要時間／金額）
- 予約一覧表示・承認・変更・キャンセル
- LINE 通知編集
- 業種テンプレートの選択（美容／整体など）

---

## 顧客側（LINE 連携）

- LIFF 起動で UUID 取得 → 顧客識別
- **直感的操作**：ドラッグ&ドロップでメニュー・日時選択
- **モダン UI**：スマホ・PC 問わず美しく操作しやすい画面
- **最小入力**：テキスト入力を極力排除、タップ・スワイプ中心
- 自動 LINE 通知による予約完了案内
- 予約履歴／キャンセル対応（開発後期）

---

## 拡張構想（v2 以降）

- 回数券／定期予約管理
- カスタムフォーム対応（アンケート等）
- 決済連携（Stripe）
- 店舗単位ロゴ／カラーカスタマイズ
- 顧客マイページ（LINE ログイン）

- **音声入力**：名前・要望の音声認識機能
- **リアルタイム通知**：予約状況のライブ更新
- **アニメーション強化**：より滑らかで直感的な UI
